## 📌 Hakkında

**Eyudio Flasher**, Arduino, ESP32, ESP32-S3 ve Arduino Leonardo/Uno gibi geliştirme kartlarına firmware yükleme işlemlerini kolaylaştırmak için geliştirilmiş modern bir yükleme aracıdır.

Karmaşık komut satırı işlemlerini tek bir arayüz altında toplayarak, kullanıcıya daha sade ve hızlı bir deneyim sunar.

Desteklenen araçlar:

- 🔹 `avrdude` → AVR tabanlı kartlar
- 🔹 `esptool` → ESP tabanlı kartlar

---

# ✨ Özellikler

✅ Modern CustomTkinter arayüzü  
✅ Otomatik COM port algılama  
✅ Arduino HEX firmware yükleme  
✅ ESP32 / ESP32-S3 BIN firmware yükleme  
✅ Kolay firmware seçimi  
✅ Gerçek zamanlı işlem çıktıları  
✅ Kullanıcı dostu hata mesajları  
✅ Taşınabilir çalışma desteği  

---

# 📦 Kurulum

## Gereksinimler

Eğer Windows işletim sistemini kullanmıyorsanız sisteminizde:

- Python 3.x
- Gerekli Python kütüphaneleri

yüklü olmalıdır.

Kütüphaneleri otomatik yüklemek için:

```bash
pip install -r requirements.txt
```

Komutu yeterlidir. 🚀

---

# ▶️ Çalıştırma

Projeyi indirdikten sonra:

```bash
python flasher.py
```

komutu ile başlatabilirsiniz.

---

# 🪟 Windows Kullanıcıları

Python kurulumu ile uğraşmak istemeyen kullanıcılar için hazırlanmış:

> 💻 Windows `.exe` sürümü

kullanılabilir.

İndir → Çalıştır → Firmware yükle 🚀

# ⚠️ Uyarılar

Firmware yükleme sırasında:

- USB bağlantısını kesmeyin.
- Doğru COM port seçtiğinizden emin olun.
- Doğru cihaz için uygun firmware kullanın.

Yanlış firmware yüklemek cihazınızın çalışmamasına sebep olabilir.
